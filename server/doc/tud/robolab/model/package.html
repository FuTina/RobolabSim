package model

Provides classes that act as model elements of the domain.

tud.robolab.model.Maze acts as represention for all kind of maze types and is managed in tud.robolab.model.MazePool. Every tud.robolab.model.Maze contains some tud.robolab.model.Point.

All requests from the same address are handled in a tud.robolab.model.Session.
